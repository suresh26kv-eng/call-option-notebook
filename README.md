# Quantitative Asset Pricing & Stochastic Simulation Framework

A comprehensive quantitative finance toolkit implemented in Python, featuring a stochastic asset path simulator and a closed-form derivative pricing engine. This project bridges stochastic calculus and computational finance to model asset dynamics and price European contingent claims.

---

## 🛠️ Core Capabilities

### 1. Stochastic Price Forecasting Framework (`Euler_Discretization.ipynb`)
* **Methodology:** Implements an **Euler-Maruyama Discretization** scheme to simulate continuous-time Geometric Brownian Motion (GBM).
* **Dynamics:** Simulates asset price paths driven by the stochastic differential equation (SDE):
  $$dS_t = \mu S_t dt + \sigma S_t dW_t$$
* **Features:** Simulates randomized daily asset drift ($\mu$) alongside unexpected market volatility shocks ($\sigma$) across thousands of independent Monte Carlo paths.

### 2. Analytical Closed-Form Engine (`BSM_Call_Option.ipynb`)
* **Methodology:** Developed a **Black-Scholes-Merton (BSM)** European Call Option pricing algorithm completely from scratch.
* **Math Foundation:** Evaluates the exact analytical price using cumulative standard normal distribution functions:
  $$C(S_t, t) = S_t N(d_1) - K e^{-r(T-t)} N(d_2)$$
* **Validation:** Serves as the analytical baseline to cross-validate the empirical payoff convergence generated by the Monte Carlo simulation paths.

---

## 📂 Repository Structure

* **`Euler_Discretization.ipynb`**: The stochastic simulation notebook containing path generation, volatility shocking, and empirical terminal distribution plots.
* **`BSM_Call_Option.ipynb`**: The closed-form analytical pricing script containing custom mathematical functions, risk-neutral valuation formulas, and model convergence cross-validation.

## ⚙️ Mathematical Framework & Libraries

This framework is built using standard quantitative computing libraries in Python:
* **NumPy:** For high-performance vectorized array operations during Monte Carlo generation.
* **SciPy (`scipy.stats`):** For precision calculations of the cumulative standard normal distribution ($N(d_1), N(d_2)$).
* **Matplotlib / Seaborn:** For visualizing the cone of simulated asset paths and terminal price distributions.
